# Frontend Quickstart for OpenCode

Use `frontend-quickstart` as a personal OpenCode skill.

## Quick Install

Clone this repository and symlink it into:

```bash
~/.config/opencode/skills/frontend-quickstart
```

Detailed steps live in:

```text
.opencode/INSTALL.md
```

## Notes

- This repository is currently distributed as a skill directory, not a custom OpenCode plugin.
- The root `SKILL.md` is the main entry point.
- Bundled helper skills under `skills/` stay available when the repository is installed as a whole.
