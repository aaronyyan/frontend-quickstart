# Frontend Quickstart for Codex

Use `frontend-quickstart` with Codex through native skill discovery.

## Quick Install

Tell Codex:

```text
Fetch and follow instructions from https://raw.githubusercontent.com/aaronyyan/frontend-quickstart/refs/heads/main/.codex/INSTALL.md
```

## What gets installed

- the root `SKILL.md`
- the bundled helper skills under `skills/`
- the analysis and validation scripts under `scripts/`

## Recommended layout

The repository should be symlinked into:

```bash
~/.agents/skills/frontend-quickstart
```

This keeps the repository self-contained and lets updates flow through a normal `git pull`.
